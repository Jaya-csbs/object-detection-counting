# Object Detection and Counting using YOLO

This project performs real-time object detection and counting using YOLOv8 and OpenCV with live webcam input.

## Features
- Live webcam object detection
- Bounding boxes with labels
- Object counting
- Real-time display

## Technologies Used
- Python
- OpenCV
- Ultralytics YOLOv8

## How to Run
```bash
pip install -r requirements.txt
python main.py
